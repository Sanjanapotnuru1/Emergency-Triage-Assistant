#  Emergency Triage Assistant

##  Overview

The Emergency Triage Assistant is an AI-powered healthcare support system that analyzes patient symptoms and generates structured medical assessments using Generative AI through the Groq API. The application helps users identify symptom severity, emergency risks, and recommended healthcare actions through an interactive Streamlit dashboard.



##  Features

- AI-powered symptom analysis  
- Emergency risk detection   
- Downloadable medical reports  
- CSV-based patient report storage  
- Responsive healthcare dashboard  



##  Technologies Used

- Python  
- Streamlit  
- Groq API  
- Pandas  
- Jupyter Notebook  
- HTML/CSS

##  Workflow

1. User enters symptoms and patient details  
2. AI analyzes symptoms using Groq API  
3. System generates healthcare assessment  
4. Emergency risks are detected  
5. Reports are displayed and stored in CSV format  



##  UI Features

- Healthcare-themed dashboard  
- Light and Dark mode  
- Responsive medical cards  
- Emergency warning alerts  
- Styled forms and upload sections
